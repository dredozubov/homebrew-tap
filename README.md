# Homebrew Tap

Homebrew formulae for [dredozubov](https://github.com/dredozubov) projects.

## Install

```bash
brew tap dredozubov/tap
brew install hazmat
```

Or in one command:

```bash
brew install dredozubov/tap/hazmat
```

## Formulae

| Formula | Description |
|---------|-------------|
| [hazmat](https://github.com/dredozubov/hazmat) | AI agent containment for macOS |
